Meet the Locals.

A one-line embed that introduces your visitors to the wildlife sharing their neighbourhood. Auto-detects the city, surfaces notable species, and illustrates each one with open Wikipedia imagery.

Live preview

This is the same widget you'd embed — running right here.

Embed it

Two short steps. No config, no API key, no build step.

1. Drop in the snippet

Paste this where you want the widget to render. It mounts to a div it creates (or to #habitat-widget-root if one already exists on the page).

<script src="https://loka.place/lokaApps/wildlife/habitat.js"></script>

2. Optionally, place the root

If you want control over position, add this div anywhere in your layout before the script:

<div id="habitat-widget-root"></div>
